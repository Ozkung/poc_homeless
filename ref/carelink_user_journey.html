เส้นทางผู้ป่วยในระบบ Care Link
จากท้องถนน → ติดตาม → ฟื้นฟู
1
พบผู้ป่วยบนท้องถนน
อาสาลงพื้นที่พบ → ถ่ายรูป + บันทึก → CM สร้างเคสใหม่ + กำหนด Triage
2
เข้ารับบริการ Mobile Clinic
ลงทะเบียน → พบหมออาสา / ทันตกรรม / นักจิต → รับยาจากเภสัชกร → หักสต็อกอัตโนมัติ
3
CM วาง Care Plan
กำหนดเงื่อนไขโรค (TB / แผล / ยาเสพติด ฯลฯ) → ระบบสร้างฟอร์ม Visit อัตโนมัติ → Assign Caregiver
4
ติดตามเยี่ยมต่อเนื่อง
Caregiver ลงพื้นที่ตาม Triage · 🔴 2 วัน / 🟡 4 วัน / 🟢 7 วัน → บันทึก Offline → Sync → ขึ้น Timeline
5
รับเข้า IPD บ้านพักพิงทางการแพทย์
CM Admit → เลือกเตียง (Bed Board) → พยาบาล/หมอดูแลรายวัน (Daily Log) → จำหน่าย + นัด F/U 1/3/6/12 เดือน
6
ฟื้นฟู / จำหน่าย / ปิดเคส
ส่งต่อสังคม / โรงพยาบาล / กลับบ้าน → บันทึก Outcome → ขึ้นรายงาน Founder

Triage & Escalation Matrix
ระดับ ความถี่เยี่ยม แจ้ง CM หลัง สูญหาย หลัง
🔴 วิกฤต ทุก 2 วัน 1 วัน 5 วัน
🟡 เฝ้าระวัง ทุก 4 วัน 2 วัน 14 วัน
🟢 ปกติ ทุก 7 วัน 3 วัน 30 วัน
🦺
อาสาลงพื้นที่ (Caregiver)
บันทึกการเยี่ยม ติดตามผู้ป่วยข้างถนน — ใช้งาน Offline ได้
เช้าวันลงพื้นที่
1
รับการแจ้งเตือนจาก LINE OA
08:00 — แจ้งรายชื่อผู้ป่วยที่ต้องเยี่ยมวันนี้ พร้อม Triage level
2
เปิดแอปผ่าน LINE → ดูรายชื่อผู้ป่วย
เห็นชื่อแฝง + สถานที่ + ระดับ Triage ของแต่ละคน (PDPA — ไม่เห็นประวัติการรักษา)
3
กรอกฟอร์ม Visit (ทำงาน Offline ได้)
Vital Signs + ถ่ายภาพ + ตอบคำถามตาม Care Plan ที่ CM กำหนด + บันทึกยาที่ส่งมอบ
4
บันทึก + Sync ข้อมูล
ข้อมูลขึ้น Timeline ผู้ป่วยทันที → CM รับ Notification อัตโนมัติ
กรณีพิเศษ
!
SOS เหตุฉุกเฉิน
กด 1 ปุ่ม → LINE alert CM ทันที + บันทึก Location อัตโนมัติ
?
หาไม่พบผู้ป่วย
บันทึกเป็น "หาไม่พบ" → นับเป็น Visit (Caregiver ไม่โดนโทษ) → CM รับแจ้ง
↑
Escalation อัตโนมัติ
เกินกำหนด → ระบบ Push แจ้ง CM → ถึง threshold → สถานะ "สูญหาย"
🏥
อาสาสมัครทางการแพทย์
หมออาสา / พยาบาล / นักจิตวิทยา — เข้าถึงประวัติการรักษา
วันงาน Mobile Clinic
1
รับแจ้ง LINE ว่าถูก Assign เข้า Event
ชื่อ Event + วันเวลา + สถานที่ + สถานีที่ต้องประจำ
2
เลือกผู้ป่วยที่จะตรวจ
เห็นเฉพาะผู้ป่วยใน Event ที่ถูก Assign เท่านั้น (PDPA)
3
ดูประวัติการรักษา (Medical Summary)
ประวัติโรค + ยาที่เคยได้รับ + Visit ที่ผ่านมา + ผลตรวจ
4
บันทึกผลการรักษา (ตาม Sub-role)
หมออาสา: วินิจฉัย + สั่งจ่ายยาได้เลยทันที → หักสต็อกยาอัตโนมัติ
พยาบาล: Vital Signs + ทำแผล + บันทึกเวชภัณฑ์ที่ใช้ → หักสต็อกเวชภัณฑ์อัตโนมัติ
นักจิต: PHQ-9 / AUDIT + ประเมิน + ส่งต่อจิตแพทย์
5
ข้อมูลขึ้น Timeline → LINE แจ้ง CM
CM รับทราบผลวินิจฉัย → อัปเดต Care Plan ตามคำแนะนำของหมอ
บ้านพักพิงทางการแพทย์ (IPD Respite House)
1
Admin Assign เข้าบ้านพักพิง
เห็นผู้ป่วยที่พักอยู่ทั้งหมด + สถานะเตียง (Bed Board)
2
กรอก Daily Log ทุกวัน
Vital Signs + อาการ + ยาที่ให้ + เวชภัณฑ์ที่ใช้ → หักสต็อกอัตโนมัติ
3
CM ออก Discharge Summary + นัดติดตาม
F/U 1 / 3 / 6 / 12 เดือน หลังจำหน่าย
🩺
Case Manager (CM)
นักสังคม — วาง Care Plan + จัดการเคส (ไม่ได้อนุมัติการรักษา)
เช้าวันทำงาน
1
เปิด Dashboard → ดู Alert ที่รอดำเนินการ
เคสเกินกำหนดเยี่ยม / รายงานจากหมออาสา / ผู้ป่วยใหม่รอ Care Plan / SOS
รับผู้ป่วยใหม่
สร้างโปรไฟล์ → ถ่ายรูป + ข้อมูล → กำหนด Triage → วาง Care Plan → Assign Caregiver → LINE แจ้ง Caregiver
รับแจ้งจากหมออาสา + บันทึกค่าใช้จ่าย
1
รับ LINE notification จากหมออาสา
หมอวินิจฉัยแล้ว → CM อัปเดต Care Plan ตามคำแนะนำ (CM ไม่ต้องอนุมัติการรักษา)
2
บันทึกค่าใช้จ่ายโรงพยาบาล (M08)
เมื่อ NPO จ่ายส่วนต่างค่ารักษา → กรอกผู้ป่วย + โรงพยาบาล + ยอดเงิน + แนบใบเสร็จ → ขึ้น F05 อัตโนมัติ
IPD Admit / Discharge
Admit ผู้ป่วย → เลือกเตียง (Bed Board) → ติดตาม Daily Log → Discharge Summary → นัด F/U 1/3/6/12 เดือน
⚙️
Admin
จัดการ Event + คลังยา + เวชภัณฑ์ + อาสาสมัคร
วันงาน Mobile Clinic
1
สร้าง Event (MC01)
กำหนดวัน + สถานที่ + สถานีที่เปิด (หมอ / ทันตกรรม / จิตใจ / เภสัช)
2
Assign อาสาทางการแพทย์เข้า Event
LINE แจ้งอาสาอัตโนมัติทุกคนที่ถูก Assign
3
ลงทะเบียนผู้ป่วยวันงาน (MC02)
Scan QR หรือค้นชื่อแฝง → ผู้ป่วยใหม่สร้างเคสได้เลย
4
ดู Event Summary หลังงาน (MC07)
จำนวนผู้รับบริการ + ยาที่จ่ายไป + ค่าใช้จ่ายรวม
จัดการคลังยา + เวชภัณฑ์ (A03)
1
รับยา / เวชภัณฑ์เข้า (Drug IN)
บริจาค: บันทึกผู้บริจาค + มูลค่า (ไม่นับเป็นค่าใช้จ่าย)
ซื้อ: ใส่เลขใบเสร็จ + ยอดเงิน + วิธีจ่าย → ขึ้นบัญชีการเงิน F05 อัตโนมัติ
2
ปรับสต็อก ADJ (กรณียาหาย / ผิดพลาด)
≤ 20 เม็ด: Admin ทำเองได้ + ระบุเหตุผล + LINE แจ้ง SA อัตโนมัติ
> 20 เม็ด: ส่ง Request ขออนุมัติ → SA Approve/Reject → Admin รับแจ้ง
👔
Founder
ดู Dashboard + รายงาน (Read-only ทั้งหมด) — เข้าผ่าน LINE Rich Menu
ทุกต้นเดือน
1
เปิด LINE OA → Rich Menu → KPI Dashboard (F01)
ผู้ป่วยรวม / เคสใหม่เดือนนี้ / อัตราการติดตาม / SOS เดือนนี้
2
ดู Analytics เชิงลึก (F02)
แผนที่ผู้ป่วย / การกระจาย Triage / Outcome สถิติ
3
ดูรายงานคลังยา (F04)
ยาคงเหลือ / ยาใกล้หมด / ประวัติการปรับ ADJ
4
ดูสรุปการเงิน (F05) → Export
ค่าซื้อยา + ค่ารักษาโรงพยาบาล + มูลค่าบริจาค (แยกประเภท) → Export PDF / Excel
