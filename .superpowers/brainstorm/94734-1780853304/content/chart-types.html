เลือกประเภทกราฟ Highcharts

กราฟแสดงการจัดกลุ่มผู้ป่วยตามความร้ายแรง (Critical / Pending / Stable)

A — Donut Chart

แสดงสัดส่วน 3 กลุ่มในวงกลม เห็นภาพรวมได้ทันที เหมาะกับการดูสัดส่วน % ของแต่ละระดับ

B — Column Chart

แท่งกราฟแยกตามกลุ่ม เปรียบเทียบจำนวนแต่ละระดับชัดเจน เหมาะถ้าต้องการดูตัวเลขสัมบูรณ์

C — Donut + Column

Donut แสดงสัดส่วน + Column แสดงจำนวน ใน card เดียวกัน ให้ข้อมูลครบใน 1 มุมมอง
