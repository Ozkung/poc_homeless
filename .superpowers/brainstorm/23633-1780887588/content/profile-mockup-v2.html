Email change flow
1. User แก้ email → กด "บันทึก"
2. Modal ขอยืนยัน password ก่อน
3. Backend ตรวจ password → update email
4. Frontend เรียก signOut() อัตโนมัติ
Avatar storage
เก็บใน apps/backend/uploads/avatars/
Serve ผ่าน GET /uploads/:filename
จำกัด 2MB, รองรับ jpg/png/webp
Docker: mount volume uploads:/app/apps/backend/uploads
