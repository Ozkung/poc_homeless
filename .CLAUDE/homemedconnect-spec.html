JWT Token Strategy
ACCESS TOKEN
  • Expiry: 15 นาที
  • Algorithm: HS256
  • Payload: sub, role, orgId
  • Transport: Authorization Header
REFRESH TOKEN
  • Expiry: 7 วัน
  • Rotation: ทุกครั้งที่ใช้
  • Storage: Redis + HttpOnly Cookie
  • Revoke: logout / reuse detected

06 — OWASP Top 10 Implementation
A01
Broken Access Control
NestJS Guard + org-scoped queries ทุก endpoint · PatientOwnerGuard · Role-based decorator
A02
Cryptographic Failures
AES-256-GCM เข้ารหัส PII · bcrypt cost=12 · HTTPS/HSTS · PostgreSQL SSL required
A03
Injection
Prisma ORM parameterized query · class-validator whitelist · ห้าม $queryRawUnsafe
A04
Insecure Design
Single-use LIFF token · Redis TTL 4hr · Check-in rate limit 1/hr · Threat model documented
A05
Security Misconfiguration
Helmet.js headers · CORS whitelist · Error masking production · .env never commit
A06
Outdated Components
npm audit CI gate · Dependabot weekly · Docker Scout image scan · Lock file committed
A07
Auth Failures
JWT rotation · LIFF server-side verify · Login rate limit 5/15min · Redis token revoke
A08
Data Integrity Failures
Line webhook HMAC-SHA256 verify · timingSafeEqual · cosign Docker image · Pin GitHub Actions SHA
A09
Logging & Monitoring
AuditInterceptor ทุก request · Threshold alert ผ่าน Line OA · Structured JSON logs
A10
SSRF
Private IP blocklist · Hostname whitelist (api.line.me เท่านั้น) · Request timeout 5s
